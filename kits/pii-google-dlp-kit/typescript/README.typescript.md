# @beddel/pii-google-dlp

Google Cloud DLP PII tokenizer adapter for the Beddel TypeScript SDK.

## Installation

```bash
pnpm add @beddel/pii-google-dlp @google-cloud/dlp
```

## Usage

```typescript
import { GoogleDLPTokenizer } from "@beddel/pii-google-dlp";

const tokenizer = new GoogleDLPTokenizer();
// Configure with Google Cloud credentials
```

## Status

Stub implementation. Full Google DLP integration planned for a future release.
